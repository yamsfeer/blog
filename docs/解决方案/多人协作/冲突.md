# 冲突

## CRDT

CRDT ( conflict-free Replicated Data Type ) 无冲突复制数据类型

* 无冲突，可复制
* 多人协作在线文档的标配方案
* CRDT 是一种数据类型，有多种具体实现，比如 Yjs、automerge、ref-crdts



* 冲突是不可避免的
* CRDT 是一种规避冲突的方案
